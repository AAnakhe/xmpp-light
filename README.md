# xmpp-light

[![Build Status](https://travis-ci.org/tumakha/xmpp-light.svg?branch=master)](https://travis-ci.org/tumakha/xmpp-light)

XMPP (Jabber) lightweight server based on Apache Vysper

### Prerequisites
Java 8, Maven 3.1+

### Run web application

    mvn spring-boot:run

### Build

    mvn clean package
